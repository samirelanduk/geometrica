geometrica
==========

matrices is a pure-python library for handling geometry and trigonometry.

Example
-------

  >>> import geometrica
  >>> geometrica.translate([(1, 1, 1), (3, 4, -8)], 5, 5, 5)
  ((6, 6, 6), (8, 9, -3))
  >>> geometrica.rotate([(1, 1, 1), (3, 4, -8)], "x", 90)
  ((1, -1, 1), (3, 8, 4))
